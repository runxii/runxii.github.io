Portfolio

Online portfolio with original design

WebsiteNext.jsReactFigmaTailwindCSS

Overview

The résumé shows what, not how. It fails to show thinking, trying, failing, and creating. This portfolio fills that gap: the decisions behind the work, a personal design sensibility, and curiosity that lives outside of formal experience. A great portfolio should work like theatre: stories, context, wins, and failures. That's why the visual language draws from the stage and the curtain.

The Challenge

Previous portfolios functioned as little more than digital résumés — projects listed, thinking omitted, and focus scattered. No signal of how problems get approached, how design decisions get made, or what growth looks like outside of formal work experience.

The Solution

  • In-Depth Case Studies: Restructured project displays into detailed case studies that articulate core pain points and strategic methodologies, demonstrating analytical and problem-solving skills to hiring managers.
  • Original UI/UX Design: Designed a distinct, theatre-themed interface using Figma, integrating an ASCII image converter tool for unique visual elements to establish the personal aesthetic.
  • "Labs" Section: Developed a dedicated module to document technical blogs and experimental mini-projects, providing concrete evidence of continuous growth and technical curiosity.

Tech Stack

  • UI Design: Applied Figma to design the aesthetic and interfaces. Implemented Asciify to create stylised decorative elements.
  • Frontend Development: Built the responsive interface using Next.js (React) and Tailwind CSS, with AI-assisted development throughout.

Outcome & Impact

  • Successfully deployed with a custom domain.
  • Achieved WCAG 2.2 AA accessibility compliance, balancing a unique visual aesthetic with high readability.
